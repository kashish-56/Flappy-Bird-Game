# 🐦 Flappy Bird Clone

A simple **Flappy Bird Clone** built using **HTML, CSS, and Vanilla JavaScript**. This project recreates the classic Flappy Bird gameplay with smooth gravity mechanics, randomly generated pipes, collision detection, score tracking, and a game over screen.

---

---

## 🚀 Features

- 🎮 Classic Flappy Bird gameplay
- 🐦 Gravity-based bird movement
- ⬆️ Jump using the **Spacebar**
- 🌿 Randomly generated top and bottom pipes
- 💥 Accurate collision detection
- 🏆 Score tracking
- 🎯 Game Over overlay with final score
- 🔄 Play Again button to restart the game
- 🎨 Custom background and realistic pipe design

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/kashish-56/flappy-bird.git
```

2. Navigate to the project folder

```bash
cd flappy-bird
```

3. Open `index.html` in your browser.

No installation or additional dependencies are required.
